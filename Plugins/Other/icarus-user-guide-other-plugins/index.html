Icarus User Guide - Other Plugins

This article is also available in: 简体中文.

This article covers some other plugins supported by Icarus 3.

Some of the following plugins are provided by ppoffice/hexo-component-inferno. Please refer to it for configuration details.

Gallery

Installation Guide

The gallery plugin of Icarus contains both lightGallery and Justified Gallery extensions. To enable it, please set plugins > gallery to true in your theme’s configuration file.

Additionally, in order to use Justified Gallery, please wrap your images with the <div class="justified-gallery"> and </div> HTML tag pair. MathJax

Installation Guide

You can use MathJax to render \(\TeX\), MathML, or AsciiMath expressions. To enable MathJax, please set plugins > mathjax to true in your theme’s configuration file.

When using the \(\TeX\) syntax, please use $ or \\( and \\) to wrap your inline expressions and and $$ or \\[ and \\] pair to wrap block expressions. You can also use \(\LaTeX\) environments directly. For example,

Outdated Browser

Installation Guide

You can use the Outdated Browser plugin to detect outdated browsers used by your site’s visitors and remind them to upgrade their browsers. To enable it, please set plugins > outdated_browser to true in your theme’s configuration file. Click here to see a live preview of this plugin.

Page Loading Animations

Installation Guide

Page Loading Animations are enabled by Icarus by default. To disable it, please set plugins > animejs to false in your theme’s configuration file.

Also, to disable the page loading progressbar, please set plugins > progressbar to false in your theme’s configuration file.
