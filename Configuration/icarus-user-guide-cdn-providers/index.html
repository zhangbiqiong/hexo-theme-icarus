Icarus User Guide - CDN Providers

This article is also available in: 简体中文.

Choosing right CDN providers can significantly reduce page loading time of your viewers. Icarus lets you to choose between several built-on CDN provider options to serve the third-party frontend libraries and asset files used by the theme.

The CDN-related functionalities of Icarus are provided by ppoffice/hexo-component-inferno. Please refer to it for a complete list of supported providers and their configurations.

Built-in CDN providers

Currently, you can choose between these built-in providers:

  • CDNs for JavaScript Libraries
    • cdnjs.com (cdnjs)
    • jsDelivr (jsdelivr)
    • UNPKG (unpkg)
    • loli.net (loli)
  • CDNs for Web Fonts
    • Google Fonts (google)
    • loli.net (loli)
  • FontAwesome Font Icon CDNs
    • FontAwesome 5 (fontawesome)
    • loli.net (loli)
